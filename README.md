Pear System Daemon
========

This is Pear System Daemon for composer


##Installation

You need to add the private repositories into your composer.json file

```json
{
    "repositories": [
        {
            "type": "vcs",
            "url":  "git@github.com:zodio/pear-system-daemon.git"
        }
    ],
    "require": {
        "zodio/pear-system-daemon": "dev-master"
    }
}
```
